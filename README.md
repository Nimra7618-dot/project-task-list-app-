# My To-Do List Web App

A simple and interactive **To-Do List web application** built with **HTML, CSS, and JavaScript**.  
This app helps you **organize your tasks**, mark them as completed, edit, or delete them easily.

---

## Features

- ✅ Add new tasks dynamically
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Completed tasks are stored separately
- ✅ Tasks are saved in **localStorage**, so they persist even after refreshing the page
